# ![NexodusLogo](https://i.imgur.com/q5YfFd9.png) Nexodus

서드파티 넥슨 게임 런쳐  


## 지원하는 게임

**현재 지원되는 게임**
 * 배틀라이트 (아레나, 로얄)  
 * 카트라이더  

**지원 예정인 게임**
 * 메이플스토리    
 * 크레이지 아케이드

 를 포함한 더 많은 게임들

## 지원하는 계정
지금 당장은 **이메일 / 아이디를 통한 로그인** 만을 지원합니다.  
QR코드 로그인, 일회용 로그인, 페이스북 및 구글 로그인 등은 지원하지 않습니다.
